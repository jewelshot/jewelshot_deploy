# Test Fixtures

## test-image.jpg

A sample image file for testing image upload functionality.

To create a test image:
1. Create any small JPG image (e.g., 500x500px)
2. Name it `test-image.jpg`
3. Place it in this directory

Or use this command to create a simple test image (requires ImageMagick):
```bash
convert -size 500x500 xc:blue test-image.jpg
```

Alternatively, just download any small image from the internet and rename it.
